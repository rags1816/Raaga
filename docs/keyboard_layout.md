```
┌─────────────────────────────────────────────────────────────┐
│  TAAR SAPTAK (Upper Octave)                                  │
│  I    O    L    9    0    -    =    ]                        │
│  Ṡ    Ṙ    Ġ    Ṁ    Ṗ    Ḋ    Ṅ    Ṡ'                      │
│  Sa'  Re'  Ga'  Ma'  Pa'  Dha' Ni'  Sa''                    │
├─────────────────────────────────────────────────────────────┤
│  MADHYA SAPTAK (Middle Octave) — main playing octave        │
│  A    S    D    F    G    H    J    K                        │

│  Sa   Re   Ga   Ma   Pa   Dha  Ni   Sa'                     │

│  Z    X    C    V    B    N    M    ,                        │

│  Sa,  Re,  Ga,  Ma,  Pa,  Dha, Ni,  Sa                      │
└─────────────────────────────────────────────────────────────┘
```

### Black keys (Komal / Tivra) — Madhya Saptak only

```
  W    E         T    Y    U
  r    g         m    d    n
  Re♭  Ga♭       Ma#  Dha♭ Ni♭
```

### Percussion / Special keys

```
  Q = Tabla Baya (Ge)
  P = Tabla Dayan (Na)
  1–8 = Octopad pads
```

## Notes

- Black (komal/tivra) keys for Mandra and Taar Saptak are click-only on screen
- All white keys across all 3 octaves have laptop hotkeys
- The layout is designed so each octave occupies a natural row:
  - Bottom row (Z–,) = Mandra
  - Home row (A–K) = Madhya  
  - Mixed (I O L, 9 0 - = ]) = Taar
