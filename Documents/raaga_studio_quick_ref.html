🚀 Quick Start Guide

  • Download raaga_studio_v4.html → open in Chrome
  • Click ▶ Start Playing — audio starts instantly (Web Audio, no CDN)
  • Turn on Tanpura Drone — keeps you in tune while playing
  • Select Shivaranjani in Raaga Highlight → only 5 green keys
  • Play only green keys → you're in the raaga!
  • Songs tab → Pehla Nasha → 🎵 Practise → Guruji guides you
  • Play-Along tab → pick song → set 50% speed → ▶ Start
  • Academy tab → ▶ Play Scale → guess the raaga → earn XP
No sound? Check tab isn't muted (right-click tab → Unmute). Needs Chrome/Edge/Safari. Headphones recommended for tabla detail.
